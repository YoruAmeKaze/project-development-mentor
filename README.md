# 项目开发教练

一步步带你把项目做出来的 AI 教练。分步骤，有条理，每一行代码都讲清楚为什么。

---

## 这个 Skill 做什么

面向编程初学者、大学生和第一次做项目的人。

开发过程中会做这些事：

- 分析项目本质，找出 MVP（最小可运行单元）
- 规划真实的开发顺序
- 推荐适合当前阶段的技术栈
- 每一步只做一件事，做完等你确认再继续
- 解释每段代码的作用和原因

目标是让你真正理解自己写出来的代码，而不只是跑起来一个你看不懂的项目。

---

## 适用场景

- 课程设计 / 大作业，不知道从哪里开始
- 有项目想法，但不清楚先做什么
- 正在学 Web 开发，想做一个完整的练手项目
- 用 AI 辅助开发，但想真正搞懂技术细节

典型项目类型：博客系统、电商平台、校园工具、管理后台、AI 工具网站。

---

## 怎么用

直接告诉 Skill 你想做什么：

```
我想做一个校园二手平台
我想做一个 AI 工具网站
我正在学 FastAPI，想做一个练手项目
```

Skill 会自动分析项目、拆解 MVP、规划顺序、推荐技术栈，然后从第一步开始带你做。

每个阶段结束后会停下来，等你说"下一步"再继续。

---

## 技术偏好

新手项目优先推荐：

| 层级 | 推荐 |
|------|------|
| 后端 | FastAPI / Flask |
| 数据库 | SQLite → MySQL |
| 前端 | Jinja2 模板 → Vue/React |
| 部署 | 等后端跑通再说 |

不会在项目初期引入 Kubernetes、微服务、分布式架构。
